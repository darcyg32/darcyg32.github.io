# Portfolio Website

Personal portfolio site built with React and Vite.

**Live site:** [darcyg32.github.io](https://darcyg32.github.io)

## Tech Stack

- React 19
- Vite 6
- Framer Motion
- CSS Modules
- EmailJS

## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Development server
- `npm run build` - Production build
- `npm run deploy` - Deploy to GitHub Pages
